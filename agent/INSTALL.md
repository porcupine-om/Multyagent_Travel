# Инструкция по установке и запуску

## Шаг 1: Активация виртуального окружения

Убедитесь, что виртуальное окружение с Python 3.10.1 активировано:

```powershell
.\venv\Scripts\Activate.ps1
```

Проверьте версию Python:
```powershell
python --version
```

Должно быть: `Python 3.10.1`

## Шаг 2: Установка зависимостей

Перейдите в папку `agent`:
```powershell
cd agent
```

Установите зависимости:
```powershell
pip install -r requirements.txt
```

## Шаг 3: Настройка переменных окружения

Убедитесь, что в корне проекта (не в папке agent) есть файл `.env` с содержимым:

```
PROXYAPI_API_KEY=your_api_key_here
PROXYAPI_BASE_URL=https://api.proxyapi.ru/openai/v1
EXCHANGERATE_API_KEY=your_exchangerate_key_here
```

**Примечание:** `EXCHANGERATE_API_KEY` опционален - можно использовать бесплатный API.

## Шаг 4: Запуск агента

Из папки `agent` запустите:

```powershell
python run.py
```

Или из корня проекта:

```powershell
python agent/run.py
```

## Примеры команд

После запуска вы можете задавать вопросы:

- "Какая погода в Москве?"
- "Сколько стоит доллар?"
- "Найди информацию о достопримечательностях Парижа"
- "Составь чек-лист для поездки в Токио с 1 по 10 января"
- "Сгенерируй QR-код для билета: ABC123"

## Устранение проблем

### Ошибка импорта модулей

Если возникают ошибки импорта, убедитесь, что вы находитесь в правильной директории и все зависимости установлены:

```powershell
pip install -r requirements.txt --upgrade
```

### Ошибка с API ключом

Проверьте, что файл `.env` находится в корне проекта и содержит правильный `PROXYAPI_API_KEY`.

### Ошибка с моделью

Если модель `gpt-5-2025-08-07` недоступна, можно изменить модель в файле `agent.py`:

```python
self.llm = ChatOpenAI(
    model="gpt-4",  # или другая доступная модель
    ...
)
```
